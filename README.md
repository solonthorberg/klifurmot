# Klifurmót

Klifurmót.is is a solution for climbing competition management for the Icelandic climbing community. Klifurmót.is handles the complete competition lifecycle from registration through real-time scoring to automated results publication, following the World Climbing standards.

**Live at [klifurmot.is](https://klifurmot.is)**

## Tech Stack

### Backend

- **Framework:** Django 5 + Django REST Framework
- **Database:** PostgreSQL
- **Auth:** JWT

### Frontend

- **Framework:** React 19
- **UI:** Tailwind v4
- **Build tool:** Vite 6
- **HTTP client:** Axios
- **State:** TanStack Query
